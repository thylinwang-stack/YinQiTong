# Design QA Checklist

## 开工前

- [ ] 已填写 design brief
- [ ] 已确认品牌气质关键词
- [ ] 已确认视觉禁区
- [ ] 已确认公开表达合规边界
- [ ] 已确认是否允许 commit / push

## Logo 结构

- [ ] 已建立画布比例
- [ ] 已建立安全区
- [ ] 已建立图形区、文字区和中轴
- [ ] 已输出深色版
- [ ] 已输出浅色版
- [ ] 已输出独立符号版
- [ ] 已检查小尺寸可读性

## 字体与文字

- [ ] 中文主名没有系统字体粗糙感
- [ ] 英文没有抢主视觉
- [ ] slogan 没有不必要标点
- [ ] slogan 没有机械撑满导致低级感
- [ ] 中文、英文、slogan 已做光学对齐
- [ ] 字体依赖已记录，定稿前考虑 outline

## 视觉边界

- [ ] 线条没有冒出文字组
- [ ] 图形没有毛边或拼接感
- [ ] 中轴间距不松散
- [ ] 左右视觉重量平衡
- [ ] 深色背景下对比度足够
- [ ] 浅色背景下对比度足够

## 交付前

- [ ] 运行 `design-qa.mjs`
- [ ] 无临时文件
- [ ] SVG 校验通过
- [ ] 关键参数符合配置
- [ ] 用户明确确认定稿
- [ ] 用户明确允许 GitHub 同步
