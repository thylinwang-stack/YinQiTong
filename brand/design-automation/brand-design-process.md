# Brand Design Process

## 1. Brief Gate

开始设计前必须先确认：

- 品牌名称
- 中文名、英文名、slogan
- 业务定位
- 目标人群
- 使用场景
- 气质关键词
- 明确禁区
- 参考对象
- 不像什么

没有 brief，不直接出图。

## 2. Direction Gate

先输出方向，不急着精修：

- 3 个气质关键词
- 3 个视觉禁区
- 2 到 3 个设计路线
- 每个路线说明适合什么、不适合什么

用户确认方向后，才进入制图。

## 3. Grid Gate

任何 Logo 或主视觉必须先建立网格：

- 画布比例
- 安全区
- 图形区
- 中文区
- 英文区
- slogan 区
- 中轴
- 留白比例
- 最小尺寸

没有网格，不进入精修。

## 4. Variant Gate

至少做三类版本：

- 几何对齐版
- 光学对齐版
- 小尺寸版

Logo 至少输出：

- 深色背景版
- 浅色背景版
- 独立符号版
- 横版主标

## 5. QA Gate

每轮修改后必须检查：

- SVG 语法
- 临时文件
- 禁用词和禁用标点
- 中文、英文、slogan 的视觉关系
- 侧标、线条和文字外框
- 深浅背景可读性
- 小尺寸可读性

## 6. Approval Gate

未获得“定稿 / 可以同步 / 可以推送”的明确指令前：

- 不 commit
- 不 push
- 不进入发布资产

用户确认定稿后，才执行 Git 提交和远端同步。
