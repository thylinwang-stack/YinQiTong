# 晤席 MEET · EAT 品牌方向

## 核心判断

`MEET · EAT` 不作为直接品牌主名，而作为英文副标。主品牌建议使用 `晤席`：

- `晤`：正式会面、商务沟通，不娱乐化。
- `席`：席位、场面、接待秩序，不餐饮化。
- 两字组合能承接高净值商务社交、城市礼宾、饭局氛围协同和餐前 brief 服务。

## 视觉母题

母题为“三席位 + 边界之环”：

- 三个点位：客户、来宾、平台礼宾助理。
- 不闭合圆环：合规边界与服务分寸。
- 中央留白：不抢戏的在场感。
- 隐约 M/E 线条：保留 meet/eat 的隐藏逻辑，但不让品牌落成餐饮 App。

## 文件

- `logo-symbol-dark.svg`：小程序图标和独立符号。
- `logo-primary-dark.svg`：深色主视觉横版。
- `logo-horizontal-light.svg`：浅色横版。
- `brandboard.html`：品牌板预览。

## 后续深化建议

1. 精修中文字标，避免过度依赖系统字体。
2. 输出黑金版、纯黑版、纯白版、单色烫金版。
3. 制作小程序启动页、用户端顶部品牌区、PC 后台登录页应用。
4. 做 32px、64px、128px、1024px 图标可读性测试。
