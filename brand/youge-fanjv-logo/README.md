# 有个饭局 Meet Eat 品牌方向

## 核心判断

`有个饭局` 比 `晤席` 更直接、更有传播力，也更符合小程序业务入口。它的风险是容易被理解成订餐、约饭或普通饭局工具，所以视觉必须负责把它抬高。

建议主品牌结构：

- 中文主名：`有个饭局`
- 英文副标：`Meet · Eat Concierge`
- 视觉写法：`M` + `eet`，`E` + `at`，突出大写 `M / E`
- 品牌口号：`体面饭局，自然发生`
- 内部定位：`高净值饭局礼宾协同`

## 视觉母题

母题为 `M/E 结构符号`：

- `M` 代表 meet，会面。
- `E` 代表 eat，共餐。
- `M` 与 `E` 共用一根结构线，表达“会面与共餐是一件事”。
- 去掉外环与席位点，让标识更克制、更像高端服务印记。
- 深色版的 `E` 改为近白色，蓝色退为辅助信息色。
- 中间圆点和底部横线已删除，避免无意义装饰干扰主符号。
- 英文副标不再使用高饱和蓝色，改为香槟金和象牙白。
- M/E 符号由圆角改为方切笔触，弱化 App 感，增强高级品牌印记。
- E 的左竖线作为完整共享主干，深色版为象牙白，避免左侧上下缺角。
- M 的右侧接入改为圆弧过渡，去掉右上方生硬尖角。
- E 的横线略加长，提高 M/E 视觉重量平衡，但不做成完全等宽。
