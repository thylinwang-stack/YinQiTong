# 微信小程序客户端

原生微信小程序 + TypeScript。当前默认使用 mock service，后续接真实后端时把 `miniprogram/app.ts` 中的 `apiMode` 改为 `real`，并在 `miniprogram/services/http.ts` 配置真实 API 域名。

## 关键能力

- 首页、服务场景、场景套餐、商务助理列表与详情
- 饭局需求表单与表单校验
- 订单确认页强制勾选服务边界协议
- 微信登录：真实模式下支付前自动调用 `wx.login`，后端换取 `openid` 并生成自定义登录态
- createPayment -> requestPayment 支付流程封装
- 支付成功、失败、取消三类结果页
- 我的订单与订单详情
- 服务协议与隐私政策

## 合规约束

- 助理详情只展示公开资料：编号、工作名、城市、风格标签、擅长场景、商务能力和合规形象照。
- 不展示助理真实手机号、微信号、身份证、住址等隐私字段。
- 客户与助理沟通通过平台客服或受控消息系统完成。
- 支付对象表达为平台服务费、预约金、餐前准备与现场协同服务费。
